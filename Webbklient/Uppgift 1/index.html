Struktur:

body håller alla element som finns i ett html-dokument. Till dessa hör text, hyperlänkar, bilder, tabeller osv.

head håller alla head-elements som tex. document, scripts, styles, meta information, osv.

html säger att det är ett html-dokument och håller alla html-komponenter.

title är titeln på html-dokumentet.


Metainformation:

meta håller den metadata som representerar html-dokumentet. Till dessa hör description, keywords, author, osv.

author Skaparen av sidan är representerad som meta-namnet authour. Ex: <meta name="author" content="Magnus Sundström">

description Beskrivning av sidan kan göras med description. Ex: <meta name="description" content="Free Web tutorials">

generator Beskrivningen av det program eller verktyg som genererat sidan. Ex: <meta name="generator" content="Brackets">

http-equiv Med denna kan man skapa vissa storheter till sidan som refreshcyklel mm. Ex: <meta http-equiv="X-UA-Compatible" content="IE=edge">

hypertext a Beskriver länkar och kan skrivas som t.ex. <a href="url">link text</a>. Denna kommer att renderas som link text.

Metainformation:

Relativ länk Denna är till skillnad från den absoluta länken relaterad till den plats som den aktuella html-filen befinner sig. Ex: <a href="../index.html">Google</a>. Denna kommer att renderas som Hem.

Absolut länk Denna länk är en url som går ända från roten och inte relativt var html-filen befinner sig i filstrukturen. Ex: <a href="http://google.com">Google</a>. Denna kommer att renderas som Google.

Ankare Ankare används för att ha länkar som scrollar webbläsaren till speciella platser längre ned eller längre upp på sidan. i>Ex: <a href="hoppa-hit">Hoppa ned till längre ned på sidan</a>. Denna kommer att renderas som Hoppa ned till längre ned på sidan.
